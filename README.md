# Bienvenido a Producciones Angel

**Producciones Angel** es una microempresa distribuidora, vendedora y productora de bolsas de aspiradora y varillas de suspensión de lavadora, además de vender repuestos de electrodomésticos,  esta empresa además de vender a diferentes tiendas enfocadas de eso, también vende a empresas grandes como CasaLimpia y Electrolux. Por lo que tiene contacto con empresas grandes, microempresas y tiendas locales.


# Problema a solucionar

La empresa tiene un registró de inventario en cuadernos, por lo que no tiene base de datos, además se le hace bastante difícil a la empresa conseguir clientes fuera de la ciudad, ya que, no tiene una página web o algún sitio al cual las personas puedan acceder a ella, es bastante difícil que pueda conseguir clientes fuera de la ciudad sin ir presencialmente. Por lo que, El problema a solucionar con esta *página de software y/o sistema de información (S.I)* es que la comercializadora tenga una base de datos donde guardar su información, se expanda nacionalmente y/o internacionalmente, además, hacer más fácil la guardada del inventario como también hacer más fáciles las ventas con un catálogo de ventas..

## Objetivo General

El objetivo general de la empresa **Producciones Angel** es elaborar un sistema de información con un registro, inventario y catálogo para subirlo a la nube, para que pueda expandirse y tener una mejor organización en las ventas.

## Objetivos específicos

1. Crear una base de datos completamente funcional.
2. Crear la CRUD tanto de administrador, cliente y empleado.
3. Hacer un inventario tanto para las ventas como para los productos.
4. Crear un catálogo de productos por categorías y precios.
5. Crear un inicio de sesión acoplado a la base de datos.
6. Crear un Registro acoplado a la base de datos.
7. Ampliar las ventas de la empresa al ya estar digitalizada.
